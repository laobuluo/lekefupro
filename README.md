# LeKeFuPro WordPress客服插件

一个专业的WordPress客服浮动图标插件，支持QQ、微信、电话、公众号等客服功能。

## 插件特点

- 支持多种客服方式：QQ、微信、电话、微信公众号
- 美观的浮动图标设计
- 响应式布局，完美适配PC和移动端
- 使用WordPress内置Dashicons图标
- 简单易用的后台设置界面
- 支持一键回到顶部功能

## 安装方法

1. 下载插件压缩包
2. 登录WordPress后台，进入"插件">"安装插件"页面
3. 点击"上传插件"，选择下载的压缩包
4. 安装完成后，点击"启用"

或者：

1. 将插件文件夹 `lekefupro` 上传到WordPress插件目录 `/wp-content/plugins/`
2. 在WordPress后台插件页面启用插件

## 使用说明

### 基本设置

1. 在WordPress后台菜单中找到"LeKeFuPro"
2. 勾选"启用插件"选项
3. 根据需要启用各项客服功能

### 客服功能设置

#### QQ客服
- 勾选"启用QQ客服"
- 上传QQ二维码图片

#### 微信客服
- 勾选"启用微信客服"
- 上传微信二维码图片

#### 电话客服
- 勾选"启用电话客服"
- 输入客服电话号码

#### 微信公众号
- 勾选"启用微信公众号"
- 上传公众号二维码图片

## 使用效果

- 在网站右侧显示浮动的客服图标
- 鼠标悬停在图标上会显示对应的二维码或信息
- 点击电话图标可以直接拨打电话
- 页面滚动时会显示"返回顶部"按钮

## 常见问题

1. **图片上传不成功？**
   - 请确保WordPress媒体库功能正常
   - 检查上传目录权限是否正确

2. **图标不显示？**
   - 确认插件已启用
   - 检查主题是否正确加载了WordPress的Dashicons

3. **手机端不能点击？**
   - 本插件完全支持移动端
   - 确保网站主题是响应式的
